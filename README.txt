To run this program, you need to first install the latest version of the python module "pronuncing".
> pip3 install pronouncing

You also need to ensure that your python version is at least 3.4 for the code to work properly.

Then, run the file "main.py".
> python3 main.py
[Assuming your current working directory is the dys/ folder. You can run the command 'pwd' to check this.]

THINGS TO NOTE:
DO NOT delete or move the preferences.json file, or the out/ folder.

To start: type "help" or "?" after you run the program.
There are preferences for the program which can be accessed with the "pref" command.
- type "? pref" to find out more.
